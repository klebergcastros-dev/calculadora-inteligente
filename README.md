# Calculadora Inteligente 🧮

Este projeto é uma calculadora interativa desenvolvida em Python para realizar operações matemáticas básicas.

## 📝 Explicação do Código
O programa utiliza:
* [cite_start]**Entrada de dados:** Recebe valores numéricos via `input()`. [cite: 1]
* [cite_start]**Conversão de tipos:** Transforma as entradas em `float` para cálculos e o resultado final em `int`. [cite: 1]
* [cite_start]**Estruturas Condicionais:** Utiliza `if/elif/else` para selecionar a operação e evitar erros como divisão por zero. [cite: 1]
* [cite_start]**Laço de Repetição:** Usa `while True` para manter o programa rodando até que o usuário decida sair. [cite: 1]

## 🐧 Como executar o arquivo .sh (Linux)
Para rodar o projeto via terminal Linux/WSL:
1. Dê permissão de execução: `chmod +x calculadora.sh`
2. Execute o script: `./calculadora.sh`
